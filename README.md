# 🛍️ Forever Store

Forever Store is a full-stack e-commerce web application for fashion retail with separate **Customer** and **Admin** interfaces. Customers can browse products, add items to cart, and make secure payments. Admins can manage products and orders through a dedicated dashboard.

## 🚀 Tech Stack

### Frontend (Customer & Admin)
- React (Vite)
- Tailwind CSS
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- Bcrypt

### Third-Party Services
- Stripe (Payment Gateway)
- Cloudinary (Image Storage)
- MongoDB Atlas (Database Hosting)


## 🔥 Key Features

- User Authentication & Authorization (JWT-based)
- Product Listing with Category Filtering
- Add to Cart & Cart Management
- Secure Online Payments using Stripe
- Order Placement & Tracking
- Admin Product Management (Add, Edit, Delete)
- Admin Order Management & Status Updates
- Cloud-based Image Upload with Cloudinary
- Fully Responsive Design (Mobile & Desktop)
